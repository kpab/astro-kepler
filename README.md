# Kepler

A versatile [Astro 7](https://astro.build) theme for blogs, portfolios, and landing pages.
Navy × orange, playful-pop design with hard offset shadows and an orbit motif.

> 🚧 Work in progress — not ready for use yet.

## Features (planned)

- 🚀 Astro 7 + Tailwind CSS v4 + TypeScript
- 📝 Blog preset — Content Collections (Markdown/MDX), tags, pagination, RSS
- 🎨 Portfolio preset — filterable project grid, case-study layout
- 🛬 Landing preset — conversion-ready sections (pricing, FAQ, testimonials)
- 🔍 SEO baked in — meta tags, OGP, sitemap
- ⚡ Zero-JS by default — vanilla `<script>` only where needed

## Getting started

```sh
pnpm install
pnpm dev
```
