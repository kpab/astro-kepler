## Full description（リッチテキストエディタ用）

Kepler is a versatile Astro 7 starter that covers three use cases — **blog,
portfolio, and landing page** — from a single, cohesive design system. Two hues
(navy × orange), two typefaces, and one set of shared tokens keep every preset
visually consistent, with a playful-pop style: hard offset shadows, slightly
rotated cards, pill buttons, and an orbit motif.

**Highlights**

- Astro 7 + Tailwind CSS v4 + TypeScript
- Landing preset — hero, logo strip, features, showcase, pricing (with toggle),
  testimonials, FAQ accordion, CTA
- Blog preset — Content Collections (Markdown/MDX), typed frontmatter, tags,
  pagination, reading time, drafts, RSS
- Portfolio preset — filterable project grid and case-study layout
- Full-text search across posts and case studies (client-side, powered by Pagefind)
- SEO baked in — canonical URLs, Open Graph, Twitter cards, sitemap.xml, RSS
- Zero-JS by default — interactions are a few lines of vanilla `<script>`; no UI framework
